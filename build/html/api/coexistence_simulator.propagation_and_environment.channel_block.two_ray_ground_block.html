coexistence_simulator.propagation_and_environment.channel_block.two_ray_ground_block

Two‑ray ground‑reflection channel block.

The model sums two propagation paths:

  1. Line‑of‑sight (LOS) path.

  2. Single ground‑reflected path with complex reflection coefficient Gamma.

Free‑space loss is represented by 1/R amplitude scaling; the reflected path is further multiplied by Gamma.

Geometry (side view):

         h_t               h_r
  Tx o----|          |----o Rx
          |          |
          |          |
-------------------------------- ground
              d

Distances

R_los = sqrt(d**2 + (h_t − h_r)**2) R_gnd = sqrt(d**2 + (h_t + h_r)**2)

Delays

tau_k = R_k / c, where c ≈ 299 792 458 m/s

Gains

g_los = 1 / R_los g_gnd = Gamma / R_gnd
