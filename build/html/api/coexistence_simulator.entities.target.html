coexistence_simulator.entities.target

Simple radar target entity.

This module defines Target, a stationary point scatterer with a constant radar cross section (RCS). The class inherits from coexistence_simulator.entities.entity_base.EntityBase so that it can be used interchangeably with other simulation entities.
